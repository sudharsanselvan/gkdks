# GKDK Digital Marketing

A polished, animated freelance studio landing page for a boutique digital marketing and development studio.

This website is designed to feel premium and motion-driven while staying lean and fast. It uses modern static HTML, CSS, and JavaScript to deliver subtle animation, interactive storytelling, and a compelling studio showcase.

## What makes this site animated?

- **Hero motion**: glowing aura and dashboard-style panel entrance.
- **Scroll reveal**: page sections fade and slide into view as you scroll.
- **Interactive hover effects**: card spotlight hover motion and mouse-following glow.
- **Ticker marquee**: infinite horizontal service ticker animates continuously.
- **Sticky header**: navigation glass effect changes when scrolling.

## Key sections

- **Hero**: premium studio positioning with a modern dashboard panel.
- **About**: explains the small-studio advantage and direct client access.
- **Core Pillars**: digital marketing, web/app development, and mobile app creation.
- **Capability Matrix**: marketing, frontend, backend, and mobile expertise.
- **Process**: five-step workflow from discovery to growth.
- **Concept work**: design direction examples instead of case studies.
- **Packages**: scoped service tiers for different client needs.
- **Contact**: WhatsApp-driven lead form and direct email contact.

## How to use

1. Open `index.html` in a modern browser.
2. The layout is fully responsive and supports desktop, tablet, and mobile screens.
3. Use the sticky nav links to jump to page sections.
4. The contact form opens WhatsApp with a pre-filled message.

## Files included

- `index.html` — main landing page markup
- `styles.css` — visual design, animation, and responsive styling
- `script.js` — scroll reveal, mobile nav, mouse glow, form behavior
- `careerspage/opportunities.html` — support page for hiring/careers

## Notes

- Replace the placeholder WhatsApp number in `script.js` with the studio's real number:
  - `WHATSAPP_NUMBER = "910000000000"`
- The site is intentionally lightweight and dependency-free.
- Use the existing structure as a launchpad for additional sections or case study pages.
